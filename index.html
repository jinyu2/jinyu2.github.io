About the Instructor

Dr. Jin Yu is a part-time lecturer at the Khoury College of Computer Sciences, Northeastern University. Holding a PhD in Electrical and Computer Engineering from Stevens Institute of Technology, he has a rich professional background, including roles such as the Director of Machine Learning at Philips/Signify Research and currently as the Head of AI at Teradyne.

Course Information

Course Overview

This course provides a hands-on introduction to the fundamentals of programming, data structures, and algorithms for data sciences, with a strong focus on Python programming.

Required Textbooks

  • Learning Python by Mark Lutz, O’Reilly Media, 2013.
  • Introduction to Algorithms, 3rd edition by Thomas Cormen et al., MIT Press, 2009.

Excerpts for required reading will be available on Piazza if not accessible online.

Supplementary Textbooks

  • Python for Data Analysis by Wes McKinney, O’Reilly Media, 2013.

Academic Integrity

Students must familiarize themselves with the university’s academic integrity policy. Plagiarism will result in no points awarded and possible reporting.

Course Topics

  • Control flow and program logic
  • Data types such as lists, dictionaries, tuples, sets
  • Data structures like arrays, queues, stacks, hash tables, trees
  • Computational complexity and more
